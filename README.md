Projeto desenvolvido utilizando ferramentas low-code, como Bubble, OutSystems, FlutterFlow e Webflow, com o objetivo de conscientizar as pessoas sobre os riscos relacionados ao uso indevido da inteligência artificial.

Além de facilitar o desenvolvimento de softwares e aplicações, as ferramentas de IA também podem ser utilizadas de forma maliciosa, como na criação de sites falsos que imitam páginas oficiais para roubo de dados pessoais (phishing: técnica de golpe digital utilizada para enganar usuários e obter informações sensíveis, como senhas, documentos e dados bancários).

Como demonstração educativa, foi desenvolvido um site inspirado na aparência de portais governamentais, simulando uma tentativa de phishing. O objetivo do projeto é alertar os usuários sobre golpes digitais, mostrando como informações sensíveis podem ser coletadas de maneira enganosa quando não há atenção à autenticidade dos sites acessados.
